1. Работа приложения.
------------------------------------------------------------------------------------------------------
-- 1.1 Сразу вы видите перед собой панель из 7 пунктов.
------------------------------------------------------------------------------------------------------
---- 1.1.1 Пункт "1. Показать таблицу" - выводит полностью всю таблицу на экран.
------------------------------------------------------------------------------------------------------
---- 1.1.2 Пункт "2. Постраничный вывод таблицы" - выводит постранично таблицу на экран.
------ 1.1.2.1 Кол-во записей для страницы - 10.
------------------------------------------------------------------------------------------------------
---- 1.1.3 Пункт "3. Вывод конкретной страницы на экран" - выводит конкретную страницу.
------ 1.1.3.1 Будет отображено существующие страницы для вывода.
------------------------------------------------------------------------------------------------------
---- 1.1.4 Пункт "4. Добавление новой записи в таблицу" - добавляет запись в конец таблицы.
------ 1.1.4.1 Поочерёдно вводите данные для каждой колонки.
------------------------------------------------------------------------------------------------------
---- 1.1.5 Пункт "5. Изменение записи в таблице" - изменяет конкретную запись в таблице.
------ 1.1.5.1 Выберите существующую запись. Затем выберите колонки для изменений и их новое значение.
------------------------------------------------------------------------------------------------------
---- 1.1.6 Пункт "6. Удаление записи в таблице" - удаляет запись из таблицы.
------ 1.1.6.1 Выберите существующую запись и удалите из таблицы.
------------------------------------------------------------------------------------------------------
---- 1.1.7 Пункт "7. Поиск по таблице" - поиск по таблице с помощью ключевых слов.
------ 1.1.7.1 Введите через пробел данные для поиска в таблице.
------------------------------------------------------------------------------------------------------
---- 1.1.8 Пункт "8. Автозаполнение таблицы" - автозаполняет таблицу разными способами.
------ 1.1.8.1 По умолчанию стоит простое заполнение с помощью не самого быстрого алгоритма.
------ 1.1.8.2 Заполнение с помощью внешнего API.
------------------------------------------------------------------------------------------------------
---- 1.1.9. Пункт "0. Настройки" - изменяет некоторые настройки приложения.
------- 1.1.9.1 Пункт "Изменить имя юзера" - изменяет имя юзера.
------- 1.1.9.2 Пункт "Изменить автозаполнение(С помощью API | Без API)" - меняет метод автозаполнения
------- 1.1.9.3 Пункт "3. Изменить кол-во записей при автозаполнении таблицы" - меняет кол-во записей
------------------------------------------------------------------------------------------------------
-- 1.2. Пользуйтесь приложением вводя команды в строку. Есть дополнительные команды который вам помогут
---- 1.2.1 Команда "q" - выходит из процесса или же завершает работу приложения если вы не в процессе.
---- 1.2.2 Команда "*" - если вы не в процессе, то выводит список комманд.
